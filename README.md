# Login and Registration System

A simple web-based Login and Registration System developed using **Python Flask**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

---

## Features

- User Registration
- User Login
- MySQL Database Integration
- Flask Routing
- HTML Templates
- CSS Styling
- Simple and Beginner-Friendly Project
- Modular Project Structure

---

## Technologies Used

- Python 3
- Flask
- MySQL
- mysql-connector-python
- HTML5
- CSS3
